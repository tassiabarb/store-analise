# Análise das Lojas Alura Store Brasil

Este projeto faz parte do desafio do curso de Data Science da Alura, onde o objetivo é analisar quatro lojas virtuais e determinar qual delas deve ser vendida pelo proprietário, Senhor João.

A análise foi realizada utilizando Python e bibliotecas de manipulação de dados e visualização no Google Colab.

---

## Objetivo

Identificar qual das quatro lojas apresenta o pior desempenho com base nos seguintes indicadores:

- Faturamento total  
- Categorias mais vendidas  
- Média das avaliações dos clientes  
- Produtos mais e menos vendidos  
- Custo médio do frete  
- Visualizações com Matplotlib

---

## Arquivos no repositório

- `analise_alura_store.ipynb` — Notebook principal com toda a análise  
- (Opcional) `dados/` — Pasta contendo os arquivos CSV das lojas  
- `README.md` — Documentação do projeto  

---

## Tecnologias utilizadas

- Python  
- Pandas  
- Matplotlib  
- Google Colab  
- GitHub  

---

## Principais resultados

- Loja com maior faturamento: **Loja 1**  
- Loja com menor faturamento: **Loja 4**  
- Melhor média de avaliações: **Loja 3**  
- Menor custo médio de frete: **Loja 4**  
- Loja recomendada para venda: **Loja 4**  

---

## Conclusão

Após análise completa dos dados, conclui-se que a **Loja 4** apresenta o pior desempenho geral entre as lojas analisadas, sendo a candidata ideal para venda.
